# 🧮 Dot Matrix Text Generator (Python)

## 📌 Description
This project is a Python-based Dot Matrix Text Generator that converts input text into a visual ASCII-style representation. It simulates a dot matrix display by mapping each character to a predefined pattern.

## 🛠️ Tech Stack
- Python
- Basic Data Structures (Lists, Functions)

## ⚙️ Features
- Converts text into dot matrix style output
- Supports uppercase alphabets (A-Z)
- Custom character mapping logic
- Clean console-based visualization

## 🧠 How It Works
- Each character is mapped to a matrix structure
- The matrix is shifted and combined dynamically
- The final output is printed using '*' symbols
